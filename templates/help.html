Help & Documentation

Admin Access

The admin interface allows you to manage the question pool. To access it:

  1. Click "Admin" in the navigation bar
  2. Enter the password: admin
  3. Click "Login"
The admin password is intentionally shared to allow community contributions, while reminding users that they are entering an area where permanent changes can be made.

Managing Questions

Editing Questions

To edit a question:

  1. Click the "Edit" button next to any question
  2. Modify any of the following fields:
    • Question text
    • Answer choices
    • Correct answer(s)
    • Explanation
    • F-tags/KSAs
    • Regulation references
  3. Click "Save Changes" to update the question
While questions cannot be deleted, you can completely edit any question you dislike into a better one. The question pool grows as these new questions are submitted to the community, so please contribute your improvements!

Updating from GitHub

To get the latest questions from our community:

  1. Click "Update Questions from Github"
  2. Read the warning about overwriting local changes
  3. Check the confirmation box if you agree
  4. Click "Update Questions"
  5. Log out and back in to see the changes in the admin interface

Backup and Restore

The application automatically maintains backups of your question pool:

  • A backup is created each time you update from GitHub
  • The system keeps your last three backups
  • You can view available backups in the Update Questions modal

To restore a previous version:

  1. Click "Update Questions from Github"
  2. Look for "Previous Versions" in the modal
  3. Click "Restore" next to the version you want
  4. Confirm the restore operation
  5. Log out and back in to see the restored questions
Restoring a backup will replace your current question pool. Make sure you want to revert to the previous version before confirming.

Sharing Your Questions

To contribute your questions back to the community:

  1. Make your desired edits to questions
  2. Click "Share Question Pool"
  3. Read and acknowledge the sharing notice
  4. Click "Share My Questions"
Your shared questions will be reviewed and may be enhanced before being added to the main question pool. Thank you for contributing to the SMQT learning community!

Best Practices

  • Always provide clear explanations for answers to help others learn
  • Include relevant regulation references when possible
  • Use F-tags and KSAs to properly categorize questions
  • Review your changes before sharing them
  • Consider updating from GitHub periodically to get new questions
